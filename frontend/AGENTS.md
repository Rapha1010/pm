Frontend Next.js app for the PM2 MVP demo Kanban.

- Next.js App Router in frontend/src/app
- Main entry: frontend/src/app/page.tsx renders AuthGate + KanbanBoard
- UI components in frontend/src/components
- State/logic helpers in frontend/src/lib
- Unit/integration tests: frontend/src/components/*.test.tsx and frontend/src/lib/*.test.ts
- E2E tests: frontend/tests (Playwright)
